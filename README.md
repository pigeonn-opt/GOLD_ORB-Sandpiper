# GOLD_ORB-Sandpiper
**Overview**
GOLD_ORB Sandpiper is an enhanced version of the GOLD_ORB Expert Advisor, featuring advanced time-filtering controls for more precise trade timing.

**Key Enhancements**
BlockedWeekDays - Specify which days of the week to disable trading (e.g., "3" for Wednesday)
BlockedHours - Specify which hours of the day to disable trading (e.g., "5,11" for 5:00 and 11:00 server time)

**Original Features**
Opening Range Breakout (ORB) strategy
Virtual trading environment for risk simulation
Dynamic position sizing based on account risk
Trailing stop management
Equity drawdown protection
